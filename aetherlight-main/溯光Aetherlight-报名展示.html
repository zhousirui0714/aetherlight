PROJECT OVERVIEW

项目概述

用现代技术重现东方美学

溯光 Aetherlight 诞生于对传统文化数字化传承的思考——如何让千年文明在数字时代焕发新生？答案是：以 AI 为核心驱动力，构建一个沉浸式的文化互动平台。

平台集成了 每日撷光、知识百科长廊、AI 雅士问答、与古人对话、AI 艺术创作 与 聊天社区 等多元功能。以大语言模型为引擎，为用户提供「每日一读、随时问道、跨越时空」的文化体验。

视觉设计汲取中国传统美学精髓：宣纸底色、墨色文字、朱砂红印章、古铜金点缀，搭配水墨晕染动效，营造出「数字书斋」的独特氛围。支持明暗双主题，让用户在不同光线下都能舒适阅读。

React 19 TanStack Router TanStack Start Tailwind CSS v4 TypeScript Vercel AI SDK Supabase Radix UI Vite Gemini 2.5 Flash Noto Serif SC

AI IMPLEMENTATION

AI 驱动核心

大语言模型如何赋能每一个功能模块

01

每日撷光 · 智能生成

基于日期自动推演节气、节日、历史事件，生成 120-180 字的诗意文化推送。结果缓存至数据库，支持前后日期浏览，让每一天都有专属的文化注脚。

generateText · JSON 结构化输出
02

问道雅士 · 流式问答

AI 雅士以清雅书卷气回答传统文化提问，250 字以内精炼作答，引用古籍原文并标注出处。非文化问题会被礼貌引导回主题，确保内容聚焦。

streamText · 实时流式响应
03

与古人对话 · 角色扮演

为每位历史人物定制系统提示词，注入朝代、风格、代表作、小传与价值观。AI 始终以古人本人的口吻对话，自然引用诗文典故，不得破坏角色。

streamText · 角色 System Prompt
INNOVATION HIGHLIGHTS

创新亮点

技术与美学的深度融合

1

东方美学视觉体系

独创「数字书斋」设计语言：宣纸底色、墨色文字、朱砂红印章、古铜金点缀。水墨晕染背景纹理、红印章组件、笔刷揭示动画、墨色晕染加载动效，将传统美学深度融入现代 UI，每一处细节都在诉说东方意韵。

2

AI 驱动的沉浸式古人对话

为 8 位历史人物精心设计性格、代表作、问候语与系统提示词。通过 AI SDK 实现流式角色扮演对话，古人以本人口吻回应，自然引用诗文典故并标注出处。支持对话历史本地持久化、长图分享生成、角色无缝切换，打造真正的「跨越时空」体验。

3

智能化每日文化推送

基于日期自动推演节气、节日、历史人物诞辰，生成富有禅意的文化推送。结果缓存至 Supabase 数据库，支持前后日期自由浏览、一键收藏与分享，让文化学习融入日常，每日撷取文明长河中的一缕光。

4

全栈现代化技术栈

React 19 + TanStack Router/Start 构建全栈 SSR 应用，Vercel AI SDK 接入 Gemini 2.5 Flash 大语言模型实现流式 AI 交互，Supabase 提供数据库与身份认证，Tailwind CSS v4 实现原子化样式，TypeScript 保障端到端类型安全。技术选型兼顾前沿性与工程成熟度。

5

完整的用户体验闭环

从浏览（每日撷光、知识长廊）、学习（AI 问道）、对话（与古人对话）到收藏、分享、历史记录，形成完整的文化体验闭环。支持用户注册登录、个人收藏管理、对话历史查询，打造个性化的数字文化空间，让每位用户都能找到属于自己的文化共鸣。
